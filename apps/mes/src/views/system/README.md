# 系统管理

## 目录说明

本目录包含系统配置和基础数据管理的功能,是MES系统的系统级配置中心。

## 功能模块

### 组织架构

- `baseOrganization.vue` - 组织架构,管理公司的组织结构
- `baseStaff.vue` - 员工管理,管理员工基础信息
- `basePosition.vue` - 岗位管理,管理岗位信息

### 用户权限

- `sysUser.vue` - 用户管理,管理系统用户和账号
- `sysRole.vue` - 角色管理,管理系统角色和权限

### 菜单与按钮

- `sysWebMenu.vue` - 菜单管理,管理系统的菜单结构
- `sysWebButton.vue` - 按钮管理,管理系统页面按钮的权限控制

### 基础配置

- `baseDictionary.vue` - 基础字典,维护系统基础字典数据
- `regionManagement.vue` - 地区管理,管理行政区划信息
