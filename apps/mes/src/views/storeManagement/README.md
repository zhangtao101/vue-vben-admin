# 仓储管理

## 目录说明

本目录包含仓储管理的完整功能,涵盖仓库管理、物料管理、库存管理和标签管理。

## 主要子模块

### 仓库管理

- `storeManage.vue` - 仓库管理,管理仓库的基础信息
- `storeBlock.vue` - 仓库区域,管理仓库的区域划分
- `storeLocation.vue` - 仓库货位,管理仓库的货位信息
- `physicalStore.vue` - 实物仓库,管理实物仓库信息

### 物料管理

- `materialManagement/` - 物料管理,包含物料管理的各项功能

### 库存管理

- `inventoryManagement/` - 库存管理,管理库存盘点和调整

### 标签管理

- `labelManagement/` - 标签管理,管理物料和产品的标签
