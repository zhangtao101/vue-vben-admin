# 作业配方

## 目录说明

本目录包含生产作业配方和工艺参数的管理功能,用于维护生产工艺和配方数据。

## 功能模块

### 配方管理

- `operationFormulaView.vue` - 作业配方视图,查看和管理作业配方
- `processFormula.vue` - 工艺配方,维护工艺配方数据
- `processFormulaInquiry.vue` - 工艺配方查询,查询和检索工艺配方
- `workstepRecipeManagement.vue` - 工步配方管理,管理工步级别的配方

### 工艺参数

- `processParameterMonitoring.vue` - 工艺参数监控,实时监控工艺参数
- `processParameterMonitoringAndQuery.vue` - 工艺参数监控与查询,监控和查询工艺参数历史数据
