# 能耗分析

## 目录说明

本目录包含能源消耗分析相关的功能模块,用于分析生产过程中的能源使用情况。

## 功能模块

- `energyFlowDirection.vue` - 能源流向分析,分析能源在生产过程中的流向和分布
- `lossAnalysis.vue` - 损耗分析,分析能源损耗情况
- `productEnergyConsumption.vue` - 产品能耗,统计和分析各产品的能源消耗
- `systemEnergyConsumption.vue` - 系统能耗,统计和分析系统的整体能源消耗
- `teamEnergyConsumptionView.vue` - 班组能耗视图,查看和分析班组能源消耗情况
