# 能耗统计

## 目录说明

本目录包含能源消耗统计相关的功能模块,用于收集和展示各类能源的统计数据。

## 功能模块

- `electricityConsumptionData.vue` - 用电量数据,统计和展示用电量信息
- `electricityCost.vue` - 电费统计,统计和分析电费数据
- `waterUsageData.vue` - 用水量数据,统计和展示用水量信息
- `gasUsageData.vue` - 用气量数据,统计和展示天然气使用量
- `collectionDataDisplay.vue` - 采集数据显示,展示采集到的能源数据
- `comprehensiveWaterFee.vue` - 综合水费,统计和分析综合水费数据
- `adjustmentFee.vue` - 调整费用,处理能源费用的调整事项
