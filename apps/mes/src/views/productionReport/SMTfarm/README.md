# SMT车间管理

## 目录说明

本目录包含 SMT 生产管理的相关功能,涵盖工单报工、计划汇总、月度汇总和日报管理。

## 功能模块

### SMT工单报工

- `SMTwork.vue` - SMT工单报工,管理 SMT 工单的报工操作

### SMT计划汇总

- `SMTfulfil.vue` - SMT计划汇总,查看 SMT 生产计划完成情况

### SMT月度汇总

- `SMTgeneral.vue` - SMT月度汇总,查看 SMT 生产月度汇总数据

### 日报管理

- `dailynewspaper/` - 日报管理,包含 SMT 生产日报相关页面
- `productionDaily.vue` - 生产日报,查看生产日报数据
