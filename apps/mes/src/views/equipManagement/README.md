# 设备管理

## 目录说明

本目录包含设备全生命周期管理的相关功能模块,涵盖设备的基础信息、监控、维护和维修。

## 功能模块

### 设备基础管理

- `energyEquipmentManagement.vue` - 能源设备管理,管理能源相关设备的基础信息
- `equipNameMaintain.vue` - 设备名称维护,维护设备的名称和分类信息

### 设备监控

- `equipmentMonitor/` - 设备监控,实时监控设备运行状态

### 设备检查与维护

- `equipmentCheck/` - 设备点检,进行设备定期检查
- `equipMaintenance.vue` - 设备维护,管理设备的维护计划和记录

### 设备维修

- `equipmentRepair.vue` - 设备维修,处理设备故障和维修工作
- `equipRecord.vue` - 设备记录,记录设备的运行和维修历史

### 资源消耗

- `resourceConsumptionEntry.vue` - 资源消耗录入,录入设备的资源消耗数据
