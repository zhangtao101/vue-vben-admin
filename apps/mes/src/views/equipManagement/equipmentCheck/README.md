# 设备点检

## 目录说明

本目录包含设备点检管理的功能,用于管理设备的点检计划、点检记录和点检项目。

## 功能模块

### 点检计划与方案

- `equipCheckPlan.vue` - 设备点检计划,制定和管理设备点检计划
- `tallyScheme.vue` - 点检方案,配置设备的点检方案
- `schemeSubsidiary.vue` - 方案明细,查看和管理点检方案的明细

### 点检记录与操作

- `checkRecord.vue` - 点检记录,记录设备的点检情况
- `checkOperation.vue` - 点检操作,执行设备点检操作

### 点检项目维护

- `tallyMaintenanceItem.vue` - 点检项目维护,维护点检项目和标准
