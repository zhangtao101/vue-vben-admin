# 设备监控

## 目录说明

本目录包含设备实时监控和历史数据查询的功能,用于监控设备运行状态和参数。

## 功能模块

### 设备信息

- `equipInfo.vue` - 设备信息,查看设备的基础信息

### 实时监控

- `parameter.vue` - 设备参数,实时监控设备运行参数
- `gather.vue` - 数据采集,查看设备实时采集的数据
- `equipStatusECLsMonitor.vue` - 设备状态ECL监控,监控设备状态和ECL数据

### 历史数据

- `monitorHistory.vue` - 监控历史,查看设备监控的历史数据
- `environmentHistory.vue` - 环境历史,查看设备环境参数的历史数据
