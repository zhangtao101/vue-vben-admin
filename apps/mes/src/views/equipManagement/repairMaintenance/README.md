# 维修维护管理

## 目录说明

本目录包含设备维修和维护相关的功能模块,涵盖故障管理、维修工单、备件管理等。

## 功能模块

### 故障管理

- `equipmentFailure.vue` - 设备故障管理,管理设备故障信息和故障树

### 维修工单

- `repairRequest.vue` - 维修申请,提交设备维修申请
- `repairOrderList.vue` - 维修工单列表,查看和处理维修工单
- `myRepairTask.vue` - 我的维修任务,查看个人维修任务

### 维修记录

- `repairRecordQuery.vue` - 维修记录查询,查询历史维修记录
- `repairBasicConfig.vue` - 维修基础配置,配置维修相关基础数据
- `repairKnowledgeBase.vue` - 维修知识库,管理维修知识文档

### 备件管理

- `sparePartsManagement.vue` - 备件管理,管理备件基础信息
- `sparePartsIoManagement.vue` - 备件出入库管理,管理备件的入库和出库
- `sparePartsInventoryMgmt.vue` - 备件库存管理,管理备件库存和报废记录
