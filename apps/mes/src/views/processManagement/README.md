# 工艺管理

## 目录说明

本目录包含工艺管理和工艺参数配置的相关功能模块,用于维护生产工艺路线和参数。

## 功能模块

### 工艺路线

- `processRoute.vue` - 工艺路线,配置和管理生产工艺路线
- `selectionOfRDProcessRoute.vue` - 研发工艺路线选择,选择和引用研发阶段的工艺路线

### 工艺参数

- `processParams.vue` - 工艺参数,维护生产工艺参数
- `processParamsQuery.vue` - 工艺参数查询,查询和检索工艺参数
- `waterContentMaintenance.vue` - 含水量维护,维护含水量相关参数

### 其他配置

- `processBase.vue` - 工艺基础,维护工艺的基础信息
- `changeOperation.vue` - 变更工序,处理工艺变更和工序调整
- `sopBinding.vue` - SOP绑定,将标准作业程序(SOP)绑定到工艺路线
