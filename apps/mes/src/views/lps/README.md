# 生产计划与统计

## 目录说明

本目录包含生产计划编制、生产统计分析、产能评估等功能模块，是MES系统中生产规划和绩效分析的核心模块。

## 主要功能模块

### 生产计划

- `schedulePlanning.vue` - 排程计划，制定和管理生产排程计划
- `preOperation.vue` - 前置作业，管理生产前的准备工作

### 产能评估

- `monthlyCapacityEvaluation.vue` - 月度产能评估，评估和分析月度产能情况
- `laborHourEvaluation.vue` - 工时评估，评估和分析工时使用情况

### 物料管理

- `kittingCheck.vue` - 齐套检查，包含3日齐套和月齐套功能，检查物料齐套情况

### 生产追踪

- `shortageTracking.vue` - 缺料追踪，跟踪和分析生产过程中的缺料情况
