# 风险管理

## 目录说明

本目录包含风险管理的完整流程,从风险识别、评估到控制和整改的全生命周期管理。

## 功能模块

### 风险评估标准

- `lecRiskAssessmentCriteria.vue` - LEC风险评估标准,维护LEC评估法的风险评价标准
- `riskStandard.vue` - 风险标准,维护风险管理的基础标准

### 风险识别与评估

- `riskIdentificationAndAssessment.vue` - 风险识别与评估,进行风险的识别、评估和分级
- `riskProfile.vue` - 风险概况,查看和统计风险的整体情况

### 风险控制

- `riskControl.vue` - 风险控制,制定和实施风险控制措施
- `riskHiddenDangerRectification.vue` - 风险隐患整改,处理风险相关的隐患整改

### 风险检查

- `riskInspection.vue` - 风险检查,进行定期的风险检查
- `riskInspectionPlanFormulation.vue` - 风险检查计划制定,制定风险检查计划
- `riskInspectionPlanExecution.vue` - 风险检查计划执行,执行风险检查计划
