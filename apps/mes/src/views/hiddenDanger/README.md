# 隐患管理

## 目录说明

本目录包含安全隐患管理的完整流程,从隐患排查到整改治理的全生命周期管理。

## 功能模块

### 隐患监控与分析

- `hiddenDangerDashboard.vue` - 隐患仪表板,提供隐患数据的可视化展示和统计分析
- `hiddenDangerReport.vue` - 隐患报表,生成和展示隐患统计报表

### 隐患排查

- `hiddenDangerInspection.vue` - 隐患排查,进行安全隐患的排查工作
- `hiddenDangerInspectionPlan.vue` - 隐患排查计划,制定和管理隐患排查计划
- `hiddenDangerInspectionTask.vue` - 隐患排查任务,管理和执行隐患排查任务
- `hiddenDangerInspectionStandard.vue` - 隐患排查标准,维护隐患排查的标准和依据

### 隐患整改

- `hiddenDangerRectification.vue` - 隐患整改,处理安全隐患的整改工作
- `hiddenDangerRectificationKanBan.vue` - 隐患整改看板,可视化展示隐患整改进度
