# 生产管理

## 目录说明

本目录包含生产操作相关的管理功能,涵盖生产入库、出库、调拨和库存管理。

## 功能模块

- `productInput.vue` - 生产入库,处理产品生产完成后的入库操作
- `productOutput.vue` - 生产出库,处理产品出库操作
- `productAllocation.vue` - 生产调拨,处理产品在不同仓库间的调拨
- `productInventory.vue` - 生产库存,查看和管理生产库存
- `disttionDetail.vue` - 分配明细,查看产品分配的详细信息
- `aintoryDetails.vue` - 库存明细,查看库存的详细数据
